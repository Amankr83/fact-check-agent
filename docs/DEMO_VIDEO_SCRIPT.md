# DEMO VIDEO SCRIPT

## 30-Second Narration

"This is FactCheck Agent, an AI-powered web app for verifying factual claims inside PDFs. I upload a document, choose how many claims to check, and start the analysis. The system extracts factual statements such as dates, percentages, financial figures, technical claims, and named entities. It then searches live web sources, collects evidence, and classifies each claim as Verified, Inaccurate, or False with a confidence score. Here, the dashboard summarizes accuracy across the document, and each evidence card shows the corrected fact plus source links. Finally, I can download a Markdown, CSV, or JSON report for review, submission, or audit."
